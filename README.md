# Physical AI & Humanoid Robotics: Embodied Intelligence in the Real World

This repository contains a comprehensive technical book on Physical AI & Humanoid Robotics, designed to bridge AI cognition with physical embodiment. The book covers ROS 2, Digital Twins, NVIDIA Isaac, Vision-Language-Action systems, and culminates in a capstone project on autonomous humanoid robots.

## Getting Started

To run the documentation site locally:

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
4. Open your browser to http://localhost:3000
